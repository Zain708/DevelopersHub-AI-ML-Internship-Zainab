# AI/ML Engineering Internship - DevelopersHub

Welcome to my AI/ML Engineering Internship portfolio! This repository contains the practical data science and machine learning projects I have completed during my internship, showcasing my progression from basic data exploration to building predictive natural language and regression models.

## Task 1: Exploring and Visualizing a Simple Dataset
* **Task objective:** Learn how to load, inspect, and visualize a dataset to understand data trends and distributions.
* **Dataset used:** Iris Dataset (via seaborn).
* **Models applied:** None (Exploratory Data Analysis only).
* **Key results and findings:** The dataset has 150 records and is perfectly clean. Petal length has a bimodal distribution, distinguishing Setosa from others. Box plots revealed outliers in the sepal width of the Setosa species.

## Task 4: Health Query Chatbot (Rule-Based vs. NLP Machine Learning)
* **Task objective:** Develop and compare a traditional rule-based chatbot against an NLP machine learning model for identifying medical intents.
* **Dataset used:** Custom text-to-intent mappings built directly into Pandas DataFrames.
* **Models applied:** Standard conditional logic vs. TF-IDF Vectorizer + Logistic Regression Pipeline (Scikit-Learn).
* **Key results and findings:** Documented a complete evolution of a conversational agent. Demonstrated how rule-based systems fail on synonyms, whereas the ML pipeline adapts to flexible sentence structures but remains constrained by its training vocabulary.

## Task 6: House Price Prediction
* **Task objective:** Build a predictive machine learning model to estimate house prices based on square footage.
* **Dataset used:** Custom synthetic real estate data (Pandas DataFrame).
* **Models applied:** Linear Regression (Scikit-Learn).
* **Key results and findings:** Successfully trained the model to find the line of best fit. The model accurately predicted the price of an unseen 2000 sqft house ($391.13k). Visualizations confirmed the model's linear trend aligns with the actual data points.
